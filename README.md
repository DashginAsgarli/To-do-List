## Overview

To-Do List App is a lightweight browser-based task manager built with vanilla JavaScript. It focuses on simplicity and usability — no frameworks, no dependencies, just clean and functional code.

---

## Features

✅ &nbsp;Add new tasks instantly  
🗑️ &nbsp;Delete tasks with one click  
🔀 &nbsp;Drag and drop to reorder tasks  
🔃 &nbsp;Sort functionality to auto-organize your list

## Getting Started

```bash
# Clone the repository
git clone https://github.com/dashginasgarli/To-do-List.git

# Open the project folder
cd To-do-List

# Run in your browser
open index.html
```

> ⚡ No installation or dependencies required — runs directly in your browser.

---

## Purpose

This project was built to practice core JavaScript skills including DOM manipulation, event handling, drag-and-drop API, and building interactive UIs from scratch — without any external libraries.
